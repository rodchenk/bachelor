# Development of a new programming language

### Variable expression:
```
x = 10;
y = 5;
x = x * 2;
z = y + x;
str_var = "Hello World";
```

### Print statement:
```
print(z + 2 + "\n"); // 27
print(str_var + "!\n"); // Hello World!
print(10 < 12); // 0 (false)
```
